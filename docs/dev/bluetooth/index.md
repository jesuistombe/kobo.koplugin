# Bluetooth Integration

This section covers the Bluetooth integration in the Kobo plugin, including dispatcher actions and
key binding configuration for Bluetooth input devices.

## Supported devices

MTK-based Kobo devices with Bluetooth support:

- Kobo Libra Colour
- Kobo Clara BW / Colour
